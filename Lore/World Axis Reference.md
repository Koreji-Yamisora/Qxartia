# World Axis Reference
_Quick reference for GMs and world-builders_

> See also: [[The Planes]] · [[The Severing]]

---

## Cosmology Overview

The **World Axis** is the cosmological model that describes how the planes of existence are structured. This model emerged after [[The Severing]] — the cataclysm that destroyed the old planar bonds when [[The Pantheon]] fell.

---

## Plane Categories

| Category | Description | Planes |
|----------|-------------|--------|
| **Prime** | The mortal world — center of the cosmology | [[Material Plane]] ([[Qxartia]]) |
| **Fundamental** | The two great realms that bound reality | [[The Astral Sea]] (above), [[The Elemental Chaos]] (below) |
| **Parallel** | Reflections of the Prime | [[The Feywild]] (bright), [[The Shadowfell]] (dark) |
| **Astral Dominions** | Realms floating in the Astral Sea | [[The Celestial]] (severed), others unexplored |
| **Elemental Realms** | Domains within the Elemental Chaos | [[The Abyss]], Undying Pyre, etc. |
| **Anomalous** | Outside the World Axis entirely | [[The Void]], [[Far Realm]] |
